# 🛒 SnapShop

**SnapShop** is a fast and modern e-commerce web application inspired by Blinkit, designed to provide a seamless online grocery shopping experience. Built using the MERN (MongoDB, Express.js, React.js, Node.js) stack, it focuses on speed, simplicity, and scalability.

---

## 🚀 Features

- 🧾 **User Authentication** – Secure login and signup using JWT.
- 🛍️ **Product Catalog** – Browse groceries by category, search products, and view details.
- 🛒 **Cart & Checkout** – Add to cart, view total, and place orders easily.
- 📦 **Order Management** – Track past and current orders in real time.
- 🧑‍💼 **Admin Dashboard** – Manage inventory, add/edit/delete products, and view orders.
- 📱 **Responsive Design** – Fully optimized for mobile and desktop.
